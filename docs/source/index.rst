================================
MyTardis Sequencing Facility App
================================

The MyTardis Sequencing Facility app ('`mytardis-seqfac`') is a component of the
MyTardis data management server which enables ingestion, storage, reporting and
sharing of high throughput sequencing data for genomics projects.

Quickstart
==========

The `mytardis-seqfac` is a `Django app <https://docs.djangoproject.com/en/1.8/ref/applications/>`_
installed as part of a MyTardis installation. The latest MyTarids documentation
can be found here: `<http://mytardis.readthedocs.org>`_

For installation instructions specific to `mytardis-seqfac`, see :doc:`installation`.
